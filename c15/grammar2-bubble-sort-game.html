15과 문법 2 보조 게임

말풍선을 알맞은 칸에 놓아 보세요.

연습 스테이지에서는 맞고 틀림을 보지 않습니다. 같은 문장 줄기를 네 추측 표현에 바꿔 넣으면서 느낌 차이를 비교합니다. 문제 스테이지부터 최적 정답, 부분 정답, 오답을 구분합니다.

연습 스테이지

말맛 지도

상대 말 내 답
-걸요

누가 말했을 때 바로 부드럽게 답해요.

안내 설명
-거예요

차분히 알려 주거나 설명할 때 잘 보여요.

내 생각
-것 같아요

내 생각이나 내 판단을 조심스럽게 말해요.

예상 뒤에 말
-텐데요

걱정이나 덧붙이는 말이 뒤에 더 올 것 같아요.

연습에서는 카드를 이 칸 저 칸 옮겨 보세요. 선택한 칸과 위의 말맛 지도가 같이 강조됩니다.

상황

움직이는 방법

1. 위 말풍선

이 말풍선이 옮길 대상이에요.

2. 아래 칸

아래 네 칸 중 한 칸으로 내려요.

3. 놓기 또는 누르기

끌어다 놓거나 칸을 눌러도 돼요.

연습에서는 여러 칸에 다시 놓으며 느낌 차이를 비교할 수 있어요.
