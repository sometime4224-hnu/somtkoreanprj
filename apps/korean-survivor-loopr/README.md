# 한국어 서바이버 루프맵 샘플 - Rapid Waves

스마트폰 세로 화면 기준의 루프맵 뱀서형 한국어 어휘 노출 게임입니다. 이 변형 버전은 한 스테이지를 더 짧고 빠르게 압축해, 강습 압박을 더 자주 느끼도록 설계했습니다.

## 이번 버전 규칙

- 한 스테이지는 `60초` 동안 진행됩니다.
- `1~3스테이지`는 `20초`, `40초`에 강습이 밀려옵니다.
- `4~5스테이지`는 `15초`, `30초`, `45초`에 강습이 밀려옵니다.
- `60초`가 지나면 남은 적을 정리한 뒤 스테이지 보스가 등장합니다.
- 총 `5개 스테이지`를 돌파한 뒤 보스러시와 최종 보스 `파멸`로 이어집니다.

## 전투 흐름

- 플레이어는 화면 중앙에 고정되고, 루프맵 배경이 움직이며 이동을 표현합니다.
- 조작은 가상 조이스틱 방식입니다.
- 적, 보스, 아이템, 강화 이름에 한국어 어휘가 계속 노출됩니다.
- 강화 카드에는 한국어와 작은 베트남어 보조 설명이 함께 표시됩니다.
- 보스 처치 후에는 전설 강화 선택이 나오고, 누적될수록 특화 효과가 강해집니다.

## 테스트 모드

- 첫 화면의 `보스러시 테스트` 버튼으로 바로 테스트할 수 있습니다.
- 테스트 모드는 `25레벨` 세팅 후 보스러시와 최종 보스전만 바로 연습합니다.
- 테스트 결과는 랭킹에 저장되지 않습니다.

## 주요 파일

- `index.html`: 진입 페이지
- `styles.css`: 모바일 레이아웃
- `src/data/vocab-data.js`: 스테이지, 적, 보상, 강화 데이터
- `src/scenes/arena-scene.js`: 전투 루프와 스테이지 진행
- `src/scenes/menu-scene.js`: 메인 화면과 랭킹
- `src/scenes/result-scene.js`: 결과 화면
