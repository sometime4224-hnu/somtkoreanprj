Grammar 2

V-(으)ㄹ 만하다

추천할 가치가 있거나 해 볼 만한 활동을 말할 때 쓰는 표현입니다. 동사 어간 뒤에 붙여 “한 번 해 볼 만하다”는 뜻을 만들고, 이유를 덧붙이면 더 설득력 있게 들립니다.

V-(으)ㄹ 만하다 진해 벚꽃 길은 가 볼 만해요.

Quick Start

먼저 추천 표현을 골라 보세요

1 / 5

Prompt

Core Pattern

해 볼 가치가 있을 때 쓰는 말

진해 벚꽃 길은 정말 가 볼 만해요.

직접 해 보면 좋겠다, 추천할 만하다, 충분히 가치가 있다는 뜻을 담습니다. 여행지, 활동, 음식, 체험을 소개할 때 특히 자주 씁니다.

주차가 조금 불편해도 풍경이 좋아서 한 번 가 볼 만해요.
1

활동을 먼저 정해요

가 보다, 타 보다, 먹어 보다처럼 실제로 해 볼 행동이 들어가면 자연스럽습니다.

2

가치를 뒷받침하는 이유를 붙여요

경치가 좋다, 시설이 편하다, 체험이 특별하다는 근거를 같이 말하면 설득력이 생깁니다.

3

권유 톤으로 마무리해요

상대에게 추천하는 장면에서는 “정말 해 볼 만해요”가 자연스럽습니다.

Extra Grammar

A/V-기로 유명하다 · N(으)로 유명하다

A/V-기로 유명하다

어떤 특징, 상태, 평가로 알려져 있을 때 씁니다.

한탄강 캠핑장은 시설이 좋기로 유명해요.

N(으)로 유명하다

대표 명사, 상징, 음식, 장소 이름으로 유명함을 말할 때 씁니다.

중국은 만리장성으로 유명해요.

고르는 기준

좋기로 유명하다는 평가를, 만리장성으로 유명하다는 대표 명사를 중심으로 말합니다.

내장산은 단풍이 아름답기로 유명해요.

Examples

Quick Tip

추천 문장을 만들 때는 이렇게 묶어요

  1. 장소/활동: 진해 벚꽃 길은, 한탄강 캠핑장은
  2. 평가: 가 볼 만해요 / 타 볼 만해요
  3. 대표 특징: 단풍이 아름답기로 유명해요 / 만리장성으로 유명해요
